Private shop operations tool

L Shop Tools

A private internal tool used by the shop owner to manage their own shop listings through Etsy Open API v3 and OAuth 2.0.

Purpose

This application is used only by the shop owner to read shop data and create or update the owner's own listings using Etsy Open API v3.

  • Uses OAuth 2.0 for private shop access.
  • Does not scrape Etsy or bypass the official API.
  • Is not offered to other sellers or third-party shops.
  • Respects API rate limits and caching guidance.

Requested access

Initial access is limited to reading shop profile data and managing the shop owner's own listings. Additional permissions will only be requested if they are required for private shop operations.
